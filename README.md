# 💹 AI Quant Trading Dashboard

An interactive financial dashboard built with **Python** and **Streamlit** for analyzing live stock data, visualizing technical indicators (RSI, EMA, SMA), and simulating trading strategies.

## 🚀 Features
- 📈 Live stock data (Yahoo Finance)
- 📊 Technical indicators (RSI, EMA, SMA)
- 📉 RSI-based backtesting simulation
- 🧠 Optional AI summary integration

## ⚙️ Tech Stack
`Python` · `Streamlit` · `pandas` · `yfinance` · `plotly` · `ta`

## 🧠 How to Run
```bash
git clone https://github.com/visha-loop/ai-quant-trading-dashboard.git
cd ai-quant-trading-dashboard
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app_trading_dashboard_clean.py
